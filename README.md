# the simplest quiz site

当プログラムは以下のリポジトリから構成されています。

- simple-quiz
- simple-quiz-api
- simple-quiz-ws
- simple-quiz-setting
- property

それぞれ、以下の役割を担います。

| リポジトリ | 役割 |
| ---- | ---- |
| simple-quiz | フロント部分(Nextjs) |
| simple-quiz-api | バックエンドのAPI部分(C#) |
| simple-quiz-ws | バックエンドのWebSocket部分(C#) |
| simple-quiz-setting | バックエンドの設定部分(Nginx) |
| property | 資産イロイロ |
